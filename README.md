# ⸸ BAPHOMET PLAYER

Reprodutor MP3 local para Android. Interface estilo Spotify em escarlate e preto.

---

## 🚀 Como gerar o APK pelo GitHub (sem instalar nada no PC)

### Passo 1 — Criar conta no GitHub
Acesse **github.com** e crie uma conta grátis.

### Passo 2 — Criar repositório
1. Clique em **"New repository"**
2. Nome: `BaphometPlayer`
3. Deixe como **Public**
4. Clique em **"Create repository"**

### Passo 3 — Subir os arquivos
Na página do repositório criado, clique em **"uploading an existing file"** e suba todos os arquivos e pastas deste projeto.

> ⚠️ A pasta `.github/` precisa ser enviada também.
> No Windows, ative "Mostrar arquivos ocultos" no Explorer para vê-la.

### Passo 4 — Aguardar a compilação
Assim que subir os arquivos, o GitHub compila automaticamente.
1. Clique na aba **"Actions"** do repositório
2. Bolinha amarela = compilando | Verde = pronto

### Passo 5 — Baixar o APK
1. Clique no build concluído (✅)
2. Role até **"Artifacts"**
3. Clique em **BAPHOMET-Player.apk**

### Passo 6 — Instalar no celular
1. Transfira o APK para o celular
2. Abra o arquivo
3. Ative "Fontes desconhecidas" se aparecer aviso
4. Instale
